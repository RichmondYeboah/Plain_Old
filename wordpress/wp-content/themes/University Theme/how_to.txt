      ____________Adding Functions_________ 
      
         ------- Adding CSS -----
      1. call the style
      2. wp_enqueue_style('', ''),
      3. call the name of the style or script wp_enqueue_style('university_main_styles', 'get_stylesheet_uri()')
      4. define the style or script 
          <? php
                function university_files() {  
                  wp_enqueue_style('university_main_styles', get_stylesheet_uri()) }
                    add_action('wp_enqueue_scripts', 'university_files');
          ?>
      
      - get_stylesheet_uri() adds the CSS with a built in wordpress function


      ------- Adding Javascript -----
      1. call the style
      2. wp_enqueue_style('', ''),
      3. call the name of the style or script wp_enqueue_scripts('main-university-js', get_theme_file_uri('js/scripts-bundled.js'), NULL, '1.0', true))
      4. define the style or script

  NOTE: Our homepage is uses index.php but if we click on one of the single post
        Wordpress will look in the theme folder for a file specifically named single.php.
        If that file doesn't exist wordpress will use index.php as a universal default fallback.
        ~ Pages are powered by index.php not single.php(Wordpress only uses the single.php file for individual post.
        for indivdual pages Wordpresslooks in our theme folder for a file named page.php )
